Titre du projet

Salah Sport - Boutique Sportive

Description

Ce projet est un site web d’une boutique sportive en ligne.
Il permet aux utilisateurs de consulter et acheter des produits sportifs comme :

vêtements de sport
chaussures
accessoires

L’objectif est de proposer une plateforme simple et rapide pour faire des achats en ligne.

Fonctionnalités
Affichage des produits
Navigation par catégories
Interface simple et facile à utiliser

Technologies utilisées
HTML
CSS
JavaScript
php
sql


Installation et utilisation
Cloner le dépôt :
git clone (https://github.com/salahsiyah2006-design/Salah-s-projet/tree/main)
Ouvrir le fichier index.html dans un navigateur.
